Controls

Click the brawler to select it. In move mode, click any reachable point to send it there.
Drag while painting to edit multiple tiles. Terrain edits redraw only the changed tile bounds plus a one-tile border.
Use mouse wheel to zoom. Use W A S D to pan the camera.
Hotkeys: 1 move, 2 land, 3 water, 4 wall, 5 erase wall.
Built as a removable standalone example under src/examples/navmesh-lab so it can ship with the repo or be removed cleanly.
